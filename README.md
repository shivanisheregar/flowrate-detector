# Bubble Flow Meter Analyzer

A computer vision-based application that automates flow-rate measurement from bubble flow meter videos.

## Features

* Bubble detection using OpenCV
* Bubble tracking
* Automatic flow-rate calculation
* Calibration mark detection (upcoming)
* OCR integration (upcoming)
* Parallel processing support
* Analytics dashboard

## Technologies

* Python
* OpenCV
* NumPy
